# Hangul Title

A small Pi extension that replaces the default terminal header with a custom Korean “파이” banner.

It shows:

- a centered gradient ASCII-art title
- a short sageuk-style greeting to a young master
- the current model and project name

The header is enabled automatically when a session starts. You can also toggle it during a session:

- `/hangul-title` — enable the custom header
- `/hangul-title-builtin` — restore Pi’s built-in header
