# Hangul Title

A small Pi extension that replaces the default terminal header with a custom Korean-inspired “파이” banner.

It shows:

- a centered gradient ASCII-art title
- a short Korean greeting
- the current model and project name

The header is enabled automatically when a session starts. You can also toggle it during a session:

- `/korean-pi-title` — enable the custom header
- `/korean-pi-title-builtin` — restore Pi’s built-in header
